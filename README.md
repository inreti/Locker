# Locker

An helper to lock users out of the Revolution manager.

Locker was PoC'ed at [MODXCCC2015](https://github.com/modx-ccc-2015)


## Goals

Prevent login into the manager when planning some maintenance (backup, migration...).

## Requirements

* PHP 5.3+
* MODX Revolution 2.3+
